# Assets

Esta carpeta debe contener los siguientes archivos para builds de producción:

| Archivo             | Tamaño recomendado | Descripción                         |
| ------------------- | ------------------- | ----------------------------------- |
| `icon.png`          | 1024×1024 px        | Ícono principal de la app           |
| `splash-icon.png`   | 1242×2688 px        | Imagen del splash screen            |
| `adaptive-icon.png` | 1024×1024 px        | Ícono adaptativo para Android       |
| `favicon.png`       | 32×32 px            | Favicon para versión web            |

Durante desarrollo con Expo Go, estos archivos son opcionales.
Podés generar assets de prueba en: https://www.appicon.co/
